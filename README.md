# DevOps Assignment

## 1. Five DevOps Concepts
1. Continuous Integration – Code ko bar-bar test aur merge karna.
2. Continuous Deployment – Code ko automatically deploy karna.
3. Version Control (Git) – Code changes ko track karna.
4. Containerization (Docker) – App ko container me pack karna.
5. Automation – Manual kaam ko tools se automatic karna.

## 2. How I Completed This Assignment

### Docker Commands Used:
docker build -t devops-assignment .
docker tag devops-assignment maulipatil2/devops-assignment:v1
docker push maulipatil2/devops-assignment:v1

### Git Commands Used:
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin <GitHub repo URL>
git push -u origin main
